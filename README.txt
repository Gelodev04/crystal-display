Crystal Display — Tailwind
=================================================

What this is
------------
A full homepage implemented with Tailwind utility classes, plus the same page broken into HTML components under /components.
The /src/index.html is a fully assembled page (no JS includes), so it is simple and production-friendly.

Build and run
-------------
1) npm install
2) npm run build
3) Open src/index.html (or serve via any static server)

Notes
-----
- Replace image placeholders with real installation photography.
- Do not introduce shadows, gradients, animations, or extra CTAs.
- Components are provided so you can move them into PHP includes, a CMS theme, or a templating layer.
